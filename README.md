# gmail-ro has moved\!

This project has been migrated to **[google-readonly](https://github.com/open-cli-collective/google-readonly)** (`gro`) with expanded support for multiple Google services.

## Install the new CLI

```bash
brew tap open-cli-collective/tap
brew install gro
```

See [google-readonly](https://github.com/open-cli-collective/google-readonly) for documentation.
